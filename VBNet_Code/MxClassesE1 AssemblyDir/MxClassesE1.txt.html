[MxClassesE1]
- [Mx.Strap, design]
- Wrapper for a private StringBuilder variable to hold a string array
- Many functions are chainable, you can say Obj.Clear().Clear().Clear() and each call returns Obj
- [Mx.Strap, operator]
- Widening Operator CType(Strap) As String = return private char array when assigned to string variable
- [Mx.Strap, function]
- Clear = chained, empties the private char array
- d(ur_text) = chained, appends the private char array
- dLine(ur_text) = chained, appends a new vbCrLf, then text
- dS(ur_text) = chained, appends single space, then text
- dSprtr(ur_sprtr, ur_text) = chained, appends separator string, then text
- HasText = tests whether the private char array is empty
- Length = size of private char array
- ToString = returns private char array as string

[MxClassesE1]
- [Mx.MxText.FileName, design]
- Wrapper for a private String variable used for holding File System paths
- Keeping System.IO.Path functions in this class makes file path traversal simples
- Many functions are chainable, you can say Obj.Clear().Clear().Clear() and each call returns Obj
- [Mx.MxText.FileName, operator]
- Widening Operator CType(FileName) As String = return private String variable when assigned to string variable
- Widening Operator CType(String) As FileName = return new FileName object when assigned to FileName variable
- [Mx.MxText.FileName, function]
- ToString = return private String variable
- d(ur_filename) = chainable, append text as File System path to private String variable 
- gParentDir = return new FileName object with parent path from private String variable
- Name = sets or returns text of right-most path entry in private String variable
- ParentDir = sets or returns text before right-most path entry in private String variable
- wAssemblyDir = chainable, overwrites private String variable with Main Module Folder path
- wParentDir = chainable, removse right-most path entry from private String variable
