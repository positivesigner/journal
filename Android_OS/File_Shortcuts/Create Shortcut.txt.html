[Chrome create a Home Screen Shortcut to Website]

- Note: You cannot create a website shortcut from an Incognito tab or a local file URL
- Click the upper-right menu (three vertical-dots) and select "Add to Home screen"



[Total Commander create a Home Screen Shortcut to Local HTML file]

- Hold-click on the HTML file and select "Create link on desktop"
- At the end of the Commend field, click the >> button and select "Choose app *"
- Select the Chrome app and click the OK / Apply button
