I can install a Chrome extension from a .CRX file

!!! Install the extension from .CRX file

[Chrome Settings page]
- [Extensions tab, top bar]
- cbox Developer Mode = set
- Click button Refresh Page

[Windows Explorer]
- [folder with .CRX file]
- Drag file onto Chrome Extensions page

[Chrome Extensions page]
- Note: The drag operation shows "Drop to Install"
- Drop the file -> Opens dialog
- Click button Add Extension
- Note: There is a new extension card shown
