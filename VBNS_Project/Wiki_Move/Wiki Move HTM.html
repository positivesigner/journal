Configuration Settings\Hard-coded settings

3rd January 2021 at 12:46am

Search pattern

  • The program is hard-coded to move files from the %USERPROFILE%\downloads folder.
  • It will only move files that match the following file search patterns
  • WikiMove_*-stamp-*.htm
  • WikiMove_*-stamp-*.html
  • WikiMove_*-stamp-*.card*.txt

  • The destination path is hard-coded between the WikiMove and -stamp- text
  • The shortest files are moved first, so any duplicate-named files like "* (1).ext" or "* (2).ext" will be moved after the original file "*.ext"
  • The program looks at the Downloads folder every five seconds
  • If another program is running with the same title as this program (another instance), this program will activate the other program's foreground window and then stop itself

Configuration Settings\Hard-coded settings

!! Search pattern
* The program is hard-coded to move files from the `%USERPROFILE%\downloads` folder.
* It will only move files that match the following file search patterns
<<<
* `WikiMove_*-stamp-*.htm`
* `WikiMove_*-stamp-*.html`
* `WikiMove_*-stamp-*.card*.txt`
<<<
!!
* The destination path is hard-coded between the `WikiMove` and `-stamp-` text
* The shortest files are moved first, so any duplicate-named files like "* (1).ext" or "* (2).ext" will be moved after the original file "*.ext"
* The program looks at the Downloads folder every five seconds
* If another program is running with the same title as this program (another instance), this program will activate the other program's foreground window and then stop itself
