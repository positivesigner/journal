GitHub repository files can be publicly browsable. [Repository files] - Create an index.html file in the root of the repository - Check-in the repository changes [Website Settings] - [Repository settings, GitHub Pages section] - Change the Source option to "master branch" - Note: GitHub's rebuild process may take a couple of minutes after changing settings or updating files - Note: The new website's URL pattern is ur_user_name.github.io/ur_repository - https://positivesigner.github.io/journal/ [Web Brower] - Navigate to your ur_user_name.github.io/ur_repository to see the index.html content
