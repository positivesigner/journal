Compare WikiFiles

22nd February 2020 at 10:08pm
DOC

You can export all the tiddler data in a TiddlyWiki to a JSON file. This way you can compare the data entered in two different TiddlyWiki files without having to look past the source code or configuration data.

  1. In the Sidebar Tools menu, select the Advanced Search button
    • This brings up the $:/AdvancedSearch tiddler
  2. On the Filter tab, enter the following text
    • [all[tiddlers]tag[INDEX]sort[title]][all[tiddlers]!tag[EXTLINK]!tag[INDEX]!tag[META]!prefix[Draft of]!is[image]!is[tag]!is[system]sort[title]]
    • Note: The Filter tab will show you a list of all tiddler titles in the TiddlyWiki file, starting with the INDEX tagged tiddler.
  3. Click the Export Tiddlers button (Up-arrow icon)
    • Note: This shows a list of export file types.
  4. Click the "JSON file" option.
    • Note: The web browser creates a "tiddlers.json" file in your downloads folder.
  5. Use Windows Explorer to rename the "tiddlers.json" file so you can remember what data it holds
  6. Make some changes in some tiddler's content
  7. Create another JSON export file
  8. Compare the two JSON export files to see the changes that were made

Create new WikiFile from template

22nd February 2020 at 10:11pm
DOC

To create a new TW File

  1. Copy the TW_Template.html as a new file name
  2. Open the file in a browser
  3. Open the Sidebar
  4. On the Tools menu, select the Control Panel button
  5. On the $:/ControlPanel tiddler, change the Title and Subtitle
    • Note: The Sidebar title and subtitle shows your new text.
  6. Close the ControlPanel tiddler
  7. Edit the INDEX tagged tiddler
  8. Change the tiddler title
  9. In the tiddler content, change the tag[META] filter to a new tag name, like tag[DOC]
    • tag[UrTagName]
  10. Save the INDEX tiddler
  11. Press Alt-N to create a new tiddler
  12. Enter a tiddler title
  13. Where you select a tag name, type in your new tag name, like DOC, and press enter
    • UrTagName
  14. Enter the tiddler content
  15. Save the tiddler
    • Note: The INDEX tiddler now shows your new tiddler title in the alphabetical list of links.
  16. In the Sidebar, click the red circle with a checkmark to save the updated TiddlyWiki file to your Downloads directory, as specified by your web browser.
  17. You can use Windows Explorer to move the updated TiddlyWiki file to the original file's directory, overwriting the previous file version.

Keyboard Shortcut

19th March 2020 at 2:37am
DOC

You can create a keyboard shortcut to run an action.

  • Create a new tiddler with the prefix "$:/config/ShortcutInfo/"

$:/config/ShortcutInfo/save-all-changes

  • In the Control Panel's Keyboard Shortcuts tab, you will find your new entry

> save-all-changes

  • Click the > arrow to see the row list of shortcut keys
    • The first row is for "All platforms"
    • Click the Pencil icon on the row to show a dialog box
      • Type a keyboard key combination, like ctrl-S
      • Click the Add Shortcut button to close the dialog box
  • Create another tiddler to hold the action commands

keyboard action SaveAllChanges

  • Add a new tag $:/tags/KeyboardShortcut
  • Enter your action commands in the text field
    • In this case, sent the "tm-save-wiki" message and set the Sidebar's visibilty variable to "no"
    • Another potential message is "tm-home", which closes all tiddlers and re-opens the default tiddlers as when you first opened the TiddlyWiki file
    • The "tm-download-file" action can export an HTML converted file of a tiddler list
<$action-sendmessage $message="tm-save-wiki"/>
<$action-setfield $tiddler="$:/state/sidebar" text="no"/>
  • Below the text field, leave the Content Type textbox empty
  • Below the Content Type field, enter the word key in the "field name" textbox, ((save-all-changes)) in the "field value" textbox, and click the Add button
  • Now the two textboxes are cleared and you see a row above them key: ((save-all-changes))
  • Save the tiddler
  • Press Ctrl-S to download a copy of the TiddlyWiki file

Linking to External Files

19th March 2020 at 6:25am
DOC

There are different kinds of file links you can create in tiddler content.

Tiddler Link

[[Create new|Create new TW File]]

Create new

When you enclose text within doubled square-brackets, it creates a Tiddler link. If you separate the text with a vertical bar ( | ), the first part is the link text and the second part is the tiddler title.

External Link

[ext[TW_Init.html]]

TW_Init.html

When you put "ext" between the openeing doubled square-brackets of a Tiddler link, the browser will open the external linke in a new tab, looking for the specified file name in the same directory as the current document.

URL Link

https://tiddlywiki.com/

https://tiddlywiki.com/

When you type an "http://" or "https://" URL, it automatically appears as an external link.

HTML Link

<a href={{TW Init Template}}  rel="noopener noreferrer" target="_blank">HTML Link</a>

HTML Link

I prefer to create the "[ext[" External link title and relative path in an EXTLINK tagged tiddler and transclude the tiddler's content.

Tiddler Content as Link Text

Create a new tiddler to hold a global macro

  • Title = mcr OneLink
  • tag = $:/tags/Macro
\define OneLink(UrParentTag)
<a target=_blank href={{$UrParentTag$}}
>$UrParentTag$</a>
\end

Create a second tiddler to hold an external link title and url

  • Title = TiddlyWiki Home Page
  • Content = https://tiddlywiki.com/

In any tiddler, call the macro and pass the title of the external link

<<OneLink "TiddlyWiki Home Page" >>

Hover your mouse over the hyperlink to see the destination URL at the bottom of the browser window

TiddlyWiki Blank Template setup

19th March 2020 at 3:30am
DOC

All the TiddlyWiki documents have the same basic features. These are in the TiddlyWiki Blank Template

_
that I use to create all new wiki documents.

  • SideBar customizations
    • The SideBar menu is hidden on start-up
    • The SideBar title is set from the $:/ControlPanel tiddler Info tab Basic sub-tab's Title and Subtitle fields
    • The SideBar menu only has Home and Save icon buttons at the top. (The SideBar menu Tools tab's checkboxes determine which icon buttons show at the SideBar top.)
  • Tiddler editing customizations
    • The "Show editor toolbar" option is cleared from the $:/ControlPanel tiddler Settings tab's Editor Toolbar section
    • Add the Text Replace bar
      _
  • Tiddler view customizations
    • The "Navigation from outside the story river" option is set to "Open at the bottom of the story river" from the $:/ControlPanel tiddler Settings tab's Tiddler Opening Behaviour section
  • There is a "Home" button in the footer of every tiddler
  • Data entry customizations
    • A Keyboard Shortcut saves the TiddlyWiki file when you press the Alt-S key combination
  • Internal documentation notes
    • The META tagged tiddlers hold "how to" information about TiddlyWiki
    • Include a Recent entries list
    • Include Font Size buttons
    • The EXTLINK tagged tiddlers hold relative or absolute URL references to the other documents
  • Default tiddler
    • There should be only one INDEX tagged tiddler showing a Table of Contents list of the first-level tiddlers
    • The INDEX tag is light-purple, set using the SideBar menu More tab' Tag Manager app
  • Browser formatting
    • If desired, you can Add a Favicon
